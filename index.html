Qmulus

Queen's University Open Data API

Documentation

Qmulus is an Open Data API providing datasets relevant to Queen's University

It is free to use for anyone to build apps and services based on the 5 different datasets. The data is collected from various university sources and is provided as-is.

Sign up for an API token here. Currently, token registration is restricted to those with an @queensu.ca email address.

Have questions about Qmulus? Reach us at contact@qmulus.io

Looking to contribute?

Most of Qmulus is open source. Help make the service better by taking a look at our contribution guide.
